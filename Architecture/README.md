# AI-Powered Learning Command Center — Mermaid Architecture Documents

This package contains Mermaid `.mmd` diagrams for the Google Cloud Platform implementation.

## Files

1. `01_system_architecture.mmd` — end-to-end GCP system architecture
2. `02_processing_pipeline.mmd` — processing pipeline from upload to dashboard
3. `03_sequence_upload_and_analysis.mmd` — sequence diagram for upload and analysis
4. `04_domain_model_er.mmd` — domain model / ER diagram
5. `05_deployment_architecture.mmd` — GCP deployment architecture
6. `06_security_iam_model.mmd` — IAM and security model
7. `07_dashboard_ui_layout.mmd` — dashboard layout
8. `08_state_machine.mmd` — learning session lifecycle state machine
9. `09_component_architecture.mmd` — application component architecture
10. `10_observability_flow.mmd` — telemetry and observability flow

These can be copied into GitHub, Mermaid Live Editor, VS Code Mermaid Preview, or project documentation.
